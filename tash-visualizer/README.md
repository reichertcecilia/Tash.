# Visuals for Tash 🌀

A psychedelic music visualizer PWA.

## Deploy to GitHub Pages (5 minutes)

1. **Create a new GitHub repo**
   - Go to [github.com/new](https://github.com/new)
   - Name it `tash-visualizer` (or anything you like)
   - Set it to **Public**
   - Click **Create repository**

2. **Upload the files**
   - Click **Add file → Upload files**
   - Drag and drop ALL files in this folder:
     - `index.html`
     - `manifest.json`
     - `sw.js`
     - `icon-192.png`
     - `icon-512.png`
   - Click **Commit changes**

3. **Enable GitHub Pages**
   - Go to **Settings → Pages**
   - Under *Source*, select **Deploy from a branch**
   - Branch: **main** / folder: **/ (root)**
   - Click **Save**

4. **Wait ~1 minute**, then visit:
   `https://YOUR-USERNAME.github.io/tash-visualizer`

## How Tash installs it

### On iPhone (Safari):
1. Open the link in Safari
2. Tap the **Share** button (box with arrow)
3. Tap **Add to Home Screen**
4. Tap **Add** — done! Opens fullscreen like a real app.

### On Android (Chrome):
1. Open the link in Chrome
2. Tap the **three dots** menu
3. Tap **Add to Home Screen** or **Install App**

### On Desktop (Chrome):
1. Open the link
2. Click the **install icon** (➕) in the address bar

## Usage
- Hit **OPEN YOUR MIND** to enable mic (reacts to music playing nearby)
- Or **JUST VISUALS** for auto-animated mode
- Click/tap anywhere to trigger a beat pulse
- Best experienced fullscreen with music playing loud 🔊
